# Installation

- Unzip the project
- Set up database at env file
    e.g,
    DB_DATABASE=votingapp
    DB_USERNAME=root
    DB_PASSWORD=1234
- Run `composer install`
- Run `npm install`
- Run `npm run production`
- Run `php artisan migrate:refresh`
- Run `php artisan serve` to serve application on `localhost:8000`
